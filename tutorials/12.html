Using R with Jupyter Notebooks for Reproducible Research

Andrie de Vries and Micheleen Harris - Microsoft

Tutorial Description

This tutorial introduces the Jupyter notebook project (previously called IPython notebooks). The tutorial will describe how Jupyter can be used for:

  • interactive coding and mark-up sandbox, sharing test code, notes (as markdown and comments) and results with colleagues, publishing snippets and useful findings
  • literate programming with interactivity, similar to like SWeave and knitr, but with added interactivity
  • Multi-language interoperability, e.g. notebooks that include R as well as Python code.
  • developing training, using Jupyter notebooks locally or in the cloud
  • delivery of training with persistent content, using a multi-tenant Jupyter notebook system

Goals

  1. Attendees will become familiar with several popular options for installing a Jupyter system on their PC's and on the cloud
  2. Show how to use R with Python as well as performing common R tasks
  3. Get attendees comfortable with using Jupyter notebooks for teaching

Tutorial Outline

  1. Introduction to Jupyter notebook structure
    1. Functions within a browser – locally, server, cloud options
    2. Good bye interpreter, hello kernel – what is a kernel?
    3. How much is R and how much is Python? – do I have to know python?
    4. Notebooks are just harmless JSON
  2. Tips
    1. Know when to download vs. save (beware of the temporary notebook!)
    2. GitHub to version control and to collaborate (they actually render notebook files on github.com)…even classrooms “pulls” down notebooks from Github
    3. Notebook files are pure JSON. You can process them if you like, however you like. For example, I’ve written a notebook for a lecture and parsed out the solutions to exercises into a separate notebook to give out later.
  3. Tricks
    1. Resizing images in an R notebook
  4. Scientific reporting in an R Jupyter notebook
    1. Comparison to the knitr system (A knitr chunk is equivalent to a Jupyter cell)
    2. Some real-life examples, showing off embedding plots, e.g. using ggplot2.
  5. Guidelines on setting up your own local Jupyter notebook system (+IRkernel)
  6. A multi-tenant Jupyter notebook system in the cloud will be used to host the entire tutorial

Background Knowledge

This tutorial requires no background knowledge of Jupyter.

We will provide an online hosted Jupyter system, but for users who want to install their own system, we’ll provide detailed download and installation instructions.
